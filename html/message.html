    Just an update to this, I spent a few minutes playing with it (as I distinctly remember list-style outside working with tables just fine) and came up with the following, which fixes the issue:

    Code: Select All
    .bbcode_list { padding-left: 20px; list-style-position: outside; }

    (if it doesn't work, just add !important to override Enjin's defaults).

    http://i.imgur.com/6hJl7Lz.png

    Works on regular forum threads as well as in PMs.
